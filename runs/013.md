## What was learned

1. **Agent concurrency requires branch awareness.** The `dateutil` formatting fix was applied by another process while this run was in progress. Rebasing and checking logs revealed the conflict was resolved.
2. **"Codecov/patch" failures often signal missing branch coverage.** Even if the logic is simple, testing both branches of a new `if` statement is necessary for project-level coverage requirements.
3. **Implicit attribute fallback in utility functions can be brittle.** `marshmallow`'s `get_value` assumed that if subscripting failed, it could safely try `getattr`, forgetting that attributes must be strings. Defensive type checking in the fallback path is essential when handling mixed-type keys.
